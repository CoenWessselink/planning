# CWS Planning UI References

Deze map bevat visuele referentieafbeeldingen voor de CWS Planning UI.

Gebruik deze afbeeldingen als visuele SSOT voor:
- desktop Apps Menu;
- mobiele bottom navigation;
- Gantt desktop/tablet/mobiel;
- Capaciteit desktop/tablet/mobiel.

Belangrijk:
- Gebruik deze afbeeldingen niet als statische UI-afbeelding in de app.
- Bouw echte componenten met bestaande routing, bestaande modules en echte data.
- Geen mockdata.
- Geen placeholders.
- Geen bestaande functionaliteit verwijderen.
- Bestaande Gantt, Capaciteit, Projectoverzicht, Instellingen en D1-logica behouden.
- Mobiel, tablet en desktop moeten allemaal bruikbaar blijven.
